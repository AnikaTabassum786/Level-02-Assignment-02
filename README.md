Features: Here is four features in this project.

1. Authentication - 
    1. User can sign up with their name,email,password,phone number as a customer or admin.
    2. User can login and get a token

2. Vehicle
   1. Admin can create vehicles.
   2. Admin and Customer can view all vehicles
   3. Admin and Customer can view specific vehicle by their ID
   4. Only admin can update vehicle information 
   5. Only admin can delete vehicle if vehicle is not booked

3. User
   1. Admin can view all users
   2. Admin can update customer's information and admin can also change another customer's role    
   3. Customer can update his own profile information, customer can not change his role.
   4. Only admin can delete user

4. Booking
   1. Admin and Customer both can create bookings.
   2. Admin can view all bookings
   3. Customer can view his own bookings
   4. Admin can update vehicle's status as returned
   5. Customer can update vehicle's status as cancelled


Technology Stack:
  1. Node.js
  2. Express.js
  3. TypeScript
  4. PostgreSQL
  5. JWT

Setup: 
 1. Node.js
 2. npm
 3. PostgreSQL
 4. env
 5. Postman
 6. git clone 'repo_url'
 7. cd 'folder_name'
 8. npm i
 9. npm run dev

 Usage Instructions:

 1. First Sign up and after login use JWT token
 2. In postman, Headers use Authorization key and value. Before paste token write Bearer
 3. For stop server, use ctrl + c
